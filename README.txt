This project makes use of the Joone and JFreeChart packages. Make sure to download them and install them in an appropriate place.

The OpenOMR application can be run using the following command:


Mac OS X / UNIX:

java -classpath PATH_TO_JOONE/joone-engine.jar:PATH_TO_JCOMMON/jcommon-1.0.5.jar:.:PATH_TO_JFREECHART/jfreechart-1.0.1.jar  -Xmx256m openomr/openomr/SheetMusic


Windows:

java -classpath PATH_TO_JOONE\joone-engine.jar:PATH_TO_JCOMMON\jcommon-1.0.5.jar:.:PATH_TO_JFREECHART\jfreechart-1.0.1.jar  -Xmx256m openomr.openomr.SheetMusic
